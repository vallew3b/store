Urban Pulse — Página web de ropa urbana

Archivos:
- index.html
- css/styles.css
- js/script.js

Instrucciones:
1. Abrir index.html en el navegador (doble clic) para ver la página localmente.
2. O bien usar un servidor simple (recomendado) desde la carpeta urban-store:

```bash
# Python 3
python -m http.server 8000
# Abrir http://localhost:8000
```

Notas:
- Las imágenes son enlaces a Unsplash como placeholders.
- Puedo añadir carrito persistente, pasarela de pago y un panel de administración si quieres.
